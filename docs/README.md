# Documentation Directory

This directory contains organized project documentation.

## Structure

### `/guides`
User-facing how-to guides and explanations:
- CSV export workflows
- Filter command reference
- Quick testing procedures
- Outlier detection explanation
- Claude plotting prompts

### `/implementation`
Technical implementation details:
- Feature implementation summaries
- System architecture notes
- Database schema documentation
- Enhancement documentation

### Archive Documentation
Historical docs from development (old implementation notes, bug reports) are in `/archive/old_docs/`.
